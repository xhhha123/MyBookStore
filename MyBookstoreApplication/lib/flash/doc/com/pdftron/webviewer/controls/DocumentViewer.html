Packagecom.pdftron.webviewer.controls
Classpublic class DocumentViewer
InheritanceDocumentViewer flash.display.Sprite

A full-featured control used for displaying a Document. The viewer supports text selection and text search, and several page view modes. Creates a new empty document viewer.

searchTextAsync()method 
public function searchTextAsync(pattern:String, mode:uint, searchNextCompleteCallback:Function):void

Searches for a particular text string on the currently displayed Document, starting on the current page (see currentPageNumber).

Parameters

pattern:String — The string to search for.
 
mode:uint — The options for search, controlling options such as case sensitivity and search direction. See text.SearchModes.
 
searchNextCompleteCallback:Function — A callback function that is called within searchTextAsync, which is used to deliver the results of the last search operation.

zoomTo()method 
public function zoomTo(zoom:Number, x:Number, y:Number):void

Adjusts the viewer's zoom factor, and positions the point (x,y) at the upper left corner of the viewer.

Parameters

zoom:Number — Horizontal position to scroll to.
 
x:Number — Vertical position to scroll to.
 
y:Number — The zoom value to which the content is zoomed to before it is scrolled. For example set zoom to 1.0 for 100%, or 2.5 for 250%.
