loadAsync()method 
public function loadAsync(partRetriever:IPartRetriever, callback:Function, cacheSize:int = 128):void

Initilizes a Document so that it can be used to load page canvases. This method must be the first method called on a new instance of a Document.

Parameters

partRetriever:IPartRetriever — A concrete instance of IPartRetriever.
 
callback:Function — A callback function that is called when the Document has been initialized. This callback receives one argument, either an Error object if the load was unsuccessful, or null if successful.
 
cacheSize:int (default = 128) — Optionally set the resource cache size, measured in megabytes. Note, this option is not compatible with all retrievers. For example streaming part retrievers usually cache all resources.

loadCanvasAsync()method 
public function loadCanvasAsync(page:uint, onLoadCanvasCallback:Function, onLinkReadyCallback:Function):void

Loads the canvas for a given page number.

Parameters

page:uint — The page number of the requested canvas.
 
onLoadCanvasCallback:Function — A callback function that is called when the canvas is loaded. Note that canvas may be missing elements, which will be added to the canvas as they are loaded.
 
onLinkReadyCallback:Function — A callback function that is called when a link is created on the canvas. The callback's argument provides access to the link, which may be used to alter the behavior of the link such as its response to mouse events.

loadPartAsync()method 
public function loadPartAsync(partName:String, onLoadPartCallback:Function, userData:Object):void

Loads the specified part. This is a low-level method that would not normally be used unless creating a custom document viewer with very specific loading requirements.

Parameters

partName:String — The name of the part, typically determined by referencing the propeties of a particular page's PageInfo object.
 
onLoadPartCallback:Function — Callback function that takes a OnLoadPartAsyncCallbackArgs and Object inputs.
 
userData:Object — Custom data that will be included in the callback function's arguments.
