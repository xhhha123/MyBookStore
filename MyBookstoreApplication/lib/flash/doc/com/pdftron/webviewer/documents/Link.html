Packagecom.pdftron.webviewer.documents
Classpublic class Link
InheritanceLink flash.display.Sprite

A link that is placed on a Document page. Links, which are used to navigate somewhere when clicked on, may point to an external URI, or a location within the document. Either navigateUrl or pageNumber will be valid, not both.
