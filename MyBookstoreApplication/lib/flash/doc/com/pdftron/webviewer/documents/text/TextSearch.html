Packagecom.pdftron.webviewer.documents.text
Classpublic class TextSearch
InheritanceTextSearch Object

TextSearch searches through a Document for a user-given search pattern. TextSearch provides users with several useful search modes and extra information besides the found string that matches the pattern. TextSearch can either keep running until a matched string is found or be set to return periodically in order for the caller to perform any necessary updates (e.g., UI updates).

public function searchBegin(pattern:String, mode:uint, startOnPageNumber:uint):void

Initilizes a new search. To perform a search, it must be followed by at least one call to searchNextAsync.

Parameters

pattern:String — The search string.
 
mode:uint — The options for the search, such as search direction and case sensitvity.
 
startOnPageNumber:uint — The page number of the Document on which to start the search.

searchNextAsync()method 
public function searchNextAsync(onTextSearchAsyncCallback:Function, userData:Object):void

Attempts to find the next instance of the search string specified in searchBegin.

Parameters

onTextSearchAsyncCallback:Function — The callback function to call when the search completes. The search may complete when the search term is found, when the entire document has been searched, and, depending on the search options, at the end of every page.
 
userData:Object — Custom data that will be included in the callback function's arguments.
