selectByRectAsync()method
public function selectByRectAsync(pageNumber:uint, startPoint:Point, endPoint:Point, onTextSelectionAsyncCallback:Function):void

Performs rectangular text selection. The selected text is what is found within the input selection points (specified by the two Point arguments), with whole words always selected.

Parameters

pageNumber:uint — The page number of the Document on which to select text.
 
startPoint:Point — The first point of the selection rectangle.
 
endPoint:Point — The second point of the selection rectangle.
 
onTextSelectionAsyncCallback:Function — The callback function that is called when the method returns.

selectByStructAsync()method 
public function selectByStructAsync(pageNumber:uint, startPoint:Point, endPoint:Point, onTextSelectionAsyncCallback:Function):void

Performs structural text selection. Which text is selected is determined using both the input selection points (specified by the two Point arguments) and the layout of the text.

Parameters

pageNumber:uint — The page number of the Document on which to select text.
 
startPoint:Point — The first point of the selection rectangle.
 
endPoint:Point — The second point of the selection rectangle.
 
onTextSelectionAsyncCallback:Function — The callback function that is called when the method returns.
