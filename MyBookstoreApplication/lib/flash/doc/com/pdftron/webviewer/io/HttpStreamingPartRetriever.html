Classpublic class HttpStreamingPartRetriever
InheritanceHttpStreamingPartRetriever flash.events.EventDispatcher
Implements IPartRetriever

Streams a .xod file hosted on a web server and display pages as they are available. This retriever (like all retrievers) is typically passed to Document's LoadAsync(IPartRetriever, Function) method or DocumentViewer's LoadAsync(IPartRetriever, Function) method to facilitate loading a Document. This particular retriever is used to retrieve a document from an HTTP server that does not support byte-serving, or optionally when the document is converted to Silverlight XPS (XOD) on demand. This retriever transfers the entire document in order from beginning to end, with pages and thumbnails displayed as each page arrives. Additionally, this retriever works with LOCAL_WITH_FILE sandbox type, which the HttpPartRetriever cannot.
